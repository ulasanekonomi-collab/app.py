# 🧠 Neuroeconomics Interactive Learning App

Aplikasi pembelajaran interaktif berbasis web untuk memahami bagaimana otak manusia mengambil keputusan ekonomi, memadukan konsep ekonomi perilaku, psikologi, dan neurosains.

## 🚀 Fitur Utama
* **Live Experiment (The Ultimatum Game):** Simulasi langsung untuk menguji bias kognitif dan melihat respons emosional otak (*Anterior Insula* vs *dlPFC*) terhadap konsep keadilan finansial.
* **Anatomi Keputusan:** Visualisasi interaktif mengenai peran *Insula Cortex*, *Prefrontal Cortex*, dan *Nucleus Accumbens* dalam menanggapi stimulus harga atau keuntungan.

## 🛠️ Teknologi yang Digunakan
* Python
* Streamlit
